## 设置快捷键

![Untitled](https://cdn.justdopay.com/notion2/md5-30a1ea6e15847286d1155ede9b0ade55.png)

在右侧输入：

```java
[
	{ "keys": ["ctrl+j"], "command": "pretty_json" }, //格式化
	{ "keys": [ "ctrl+n" ], "command": "un_pretty_json" }  //压缩
]
```
