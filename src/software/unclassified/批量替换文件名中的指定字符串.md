date: 2022-12-30

## 方法一
比较通用
```shell
find ./ -name "*keyword*" | while read f; do mv $f ${f/keyword/targetword}; done
```
命令解释：找出文件名中有 keyword 字符串的文件，然后通过mv命令把 keyword替换为targetword
