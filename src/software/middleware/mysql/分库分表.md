分表：垂直拆分和水平拆分

分库：垂直拆分（一个库有A-Z 26张表，可以分到两个库：A-K，K-Z）

需要注意的一点是：分表仅仅是解决了单一表数据过大的问题，但由于表的数据还是在同一台机器上，其实对于提升MySQL并发能力没有什么意义，所以 水平拆分最好分库 。
